# Versão Estática - Presentation Eval

Esta é uma versão recriada do projeto Presentation Eval utilizando HTML, CSS (Tailwind via CDN) e JavaScript puro.

## Como rodar

Basta abrir o arquivo `index.html` em qualquer navegador moderno.

Não é necessária instalação de dependências (npm install) ou build.

## Estrutura

- `index.html`: Arquivo principal.
- `script.js`: Contém toda a lógica de navegação, renderização e gráficos.
- `logo.png`: Logo da aplicação.

## Notas

- A gravação de vídeo é simulada, assim como no projeto original.
- O gráfico utiliza Chart.js para renderização.
- Os ícones utilizam a biblioteca Lucide.
